Colors Cheatsheet

Types of Colors

Value Type Description
green Color keywords Simple and easy! You can use the name of the color you are looking for instead of a value; however, it has to be a color keyword CSS knows. The amount of color keywords changes from different CSS versions, so while this type is straightforward, it can be very limiting.
#008000 Hex Values Hex can look pretty confusing at first but can be broken down into red, green, and blue using hexadecimal values. Each digit can range from 1-255 using hex(15) values 0-9 and A-F. So, for example, the Hex value of #008000 is just 00 for Red, 80 which translates to 120 for Green and 00 for blue. A significant benefit of this system is that hex values are nice and compact, and if all three parts of the hex value are the same double digits, for example, black #000000 can be shortened to #000.
rgb(0,128,0) RGB Values Three values make up RBG: Red, Green, and Blue. These values range from 0-255. Where 0 is no amount of that color and 255 is the max amount of that color. The primary benefit of this type is that you can get an exact color since you control the amount of red, green, and blue used for your color and make slight changes by modifying the easy-to-understand variables!
hsl(120,100%,25%) HSL Values Three values make up HSL: hue, saturation, and light. It can be a little confusing at first, but once masted can give you a lot of control over how a color can look. The first aspect of the HSL value, hue, is a value that ranges from 0-360. Each value represents a degree on a color wheel, 0 being red, 120 being green, 240 being blue, and 360 being red again. The second aspect is saturation; this percentage value represents how saturated the color ranges from 0%-100%. The lower the color becomes, the more grey the color. Finally, the last aspect is Light, this is a percentage value ranging from 0%-100%, and this affects how light the color is, with 0% being the darkest and 100% being the lightest. Using this as your color value gives you an aspect of customization that other values don't have.
hwb(120,0%,50%) HWB Value HWB values are similar to HSL because both use hue values. The main difference is instead of saturation and light, you have white and black. White and black are percentage values that range from 0%-100%, with the higher their percentage, the more white or black the color is, respectively. Note that both white and black color values should not equal more than 100%. When both do equal 100%, your color will appear grey, a mix of only black and white. HWB is generally easier to understand and change than HSL while providing similar benefits.

Color Opacity

Value Type Description
#00800088 Hex Value with Alpha To change your opacity with hex values is simple! Add two digits that range from 00-FF at the end of your original hex value. These digits change a value called alpha, with 00 completely clear and FF being completely opaque.
rgba(0,128,0,0.5) RGBA Value RGBA is similar to RGB with the addition of A(alpha) value that changes the opacity of color from 0-1. 0 makes the color completely clear, and 1 makes the color completely opaque.
hsla(120,100%,25%,0.5) HSLA Value HSLA is just like HSL values but with the additional A(alpha) value that changes the opacity of the color that ranger 0-1. 0 makes the color completely clear, and 1 makes the color completely opaque.
hwba(120,0%,50%, 0.5) HWBA Value HWBA is just HWB values but with the additional A(alpha) value that changes the opacity of the color that ranger 0-1. 0 makes the color completely clear, and the 1 makes the color completely opaque.
